# Dutch Address

A comprehensive Python library for validating, correcting, and looking up Dutch postal addresses.

## Features

- **Address Validation**: Validate Dutch addresses against official postal data
- **Address Correction**: Correct mistyped street and city names using fuzzy matching
- **Address Parsing**: Parse addresses from standard two-line format
- **Partial Address Lookup**: Find addresses based on partial information
- **REST API** (Optional): JSON API for address operations
- **High Performance**: Optimized data structures and caching
- **Type Annotated**: Full type hints for better IDE support
